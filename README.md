# Lab 4: Testing & Automation (Backend)

Цей проект присвячений впровадженню автоматизованого тестування та інструментів збору даних для REST API на базі FastAPI.

##  Основні можливості

*   **Модульне тестування:** Використання `pytest` для перевірки бізнес-логіки.
*   **Code Coverage:** Покриття коду тестами становить **63%**.
*   **Performance Testing:** Симуляція навантаження на ендпоінти за допомогою `Locust`.
*   **Web Scraping:** Автоматизація браузера через `Selenium` для збору даних з фронтенду.

##  Технологічний стек

*   **Backend:** Python 3.x, FastAPI, Uvicorn.
*   **Testing:** Pytest, Pytest-cov, Locust.
*   **Automation:** Selenium, Webdriver-manager.
